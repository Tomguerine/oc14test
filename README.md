# HRnet

Welcome to HRnet! This is our company's internal application to create and view employee records.

## Development

All application code lives in the `hrnet-react` directory and is written in TypeScript.

### Install

```bash
cd hrnet-react
npm install
```

### Run

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Test

```bash
npm run lint
```
